Register an Admin:

Method: POST

URL: http://localhost:5000/api/auth/register

{
    "name": "Admin2",
    "email": "admin2@example.com",
    "password": "password123",
    "role": "admin"
}

Register an User:

Method: POST

URL: http://localhost:5000/api/auth/register

{
    "name": "User2",
    "email": "user2@example.com",
    "password": "password123",
    "role": "admin"
}

Log in as User/Admin:
Set up your request to log in as a user:

Method: POST

URL: http://localhost:5000/api/auth/login

{
    "email": "user2@example.com",
    "password": "password123"
}

Upload an Assignment:

Method: POST

URL: http://localhost:5000/api/assignments/upload

Headers: Add a header for x-auth-token and paste the token you copied.

{
    "task": "Hello World",
    "admin": "admin2@example.com"
}
Click Send.

View Assignments as Admin:

Method: GET

URL: http://localhost:5000/api/assignments

Headers: Add a header for x-auth-token and paste the admin token.

Click Send.

Accept or Reject an Assignment:
Accept an Assignment:

Method: POST

URL: http://localhost:5000/api/assignments/{id}/accept

Headers: Add a header for x-auth-token and paste the admin token.

Click Send.

Reject an Assignment:

Method: POST

URL: http://localhost:5000/api/assignments/{id}/reject 

Headers: Add a header for x-auth-token and paste the admin token.

Click Send.
